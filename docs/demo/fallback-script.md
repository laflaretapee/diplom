# Fallback Script

## Если WebSocket ведёт себя нестабильно

- Не строить показ вокруг live-событий.
- Вести сценарий через:
  - `/queue` для смены статуса;
  - `/orders/history` для подтверждения результата;
  - при необходимости ручной refresh страницы.
- Основная фраза:
  - «Даже если realtime-канал недоступен, статусный поток и история заказов продолжают работать через backend API».

## Если AI assistant отвечает нестабильно

- Не тратить время на отладку.
- Пропустить `/assistant` и перейти к `/dashboard`.
- Показать вместо этого:
  - summary;
  - revenue;
  - channels;
  - top dishes;
  - при необходимости forecast/anomalies через Swagger, если просят показать API-уровень.
- Основная фраза:
  - «ИИ здесь бонусный слой над аналитикой. Базовая бизнес-ценность системы — в заказах, складе, аналитике и франчайзи-контуре».

## Если session refresh ломается в браузере

- Открыть новую вкладку и войти заново под нужной ролью.
- Использовать фиксированные demo-аккаунты:
  - `admin@japonica.example.com / Admin1234!`
  - `manager-ufa1@japonica.example.com / Demo1234!`
  - `staff-ufa1@japonica.example.com / Demo1234!`
  - `franchisee-ufa1@japonica.example.com / Demo1234!`

## Если warehouse-flow не подходит для live-операции

- Не делать приход и корректировку вживую.
- Показать уже существующие:
  - остатки;
  - low-stock;
  - историю движений;
  - audit trail.
- Основная фраза:
  - «Операции поддерживаются, но для экономии времени я покажу уже созданную историю и состояние остатков».

## Если franchisee board подтормаживает

- Открыть одну готовую карточку франчайзи.
- Сфокусироваться на:
  - стадии;
  - задачах;
  - заметках;
  - привязанных точках.
- Не пытаться обойти всю доску колонка за колонкой.

## Минимальный маршрут, если остаётся 3–4 минуты

1. Вход под `manager-ufa1`.
2. `/queue` → смена статуса заказа.
3. `/warehouse` → остатки и движения.
4. Вход под `admin`.
5. `/dashboard` → summary и выручка.
6. `/franchisee` → одна карточка франчайзи.

Этот маршрут покрывает главную бизнес-ценность даже без AI и без акцента на realtime.
