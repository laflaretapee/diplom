# Japonica CRM Frontend

Minimal React 18 + TypeScript + Vite scaffold for the Japonica CRM UI.

## Included

- React Router based auth flow
- React Query provider
- Zustand auth store placeholder
- Axios client placeholder
- Ant Design layout and theme baseline
- Role-aware shell with login and dashboard placeholders

## Local development

```bash
npm install
npm run dev
```

Set `VITE_API_BASE_URL` if the API is not served from `/api`.
